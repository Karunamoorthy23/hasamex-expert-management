n8n Integration

Project → Expert Search → Email Invites

End-to-end n8n workflow for the Expert Management app

Executive Architecture Overview

System Overview

1. Trigger

Creating a Project in the app posts minimal project context to an n8n webhook.

2. Orchestration

n8n calls back the app to execute a “filter‑first” LinkedIn search via Apify.

3. Outreach

n8n (or the app) triggers invitation emails to lead experts with a link to the expert form.

High-Level Flow

Core Endpoints & Security

Authentication

Protected with N8N_SERVICE_TOKEN.

Header: Authorization: Bearer <token>
Query: ?token=<token>

Key Endpoints Map

  • Create Project: projects.py
  • n8n Status: n8n_webhook.py
  • Search Experts: n8n_webhook.py
  • Send Emails: n8n_webhook.py
  • Email Sender: projects.py

End‑to‑End Flow: Part 1

1) Create a New Project

  • Frontend posts to Project API; app notifies n8n webhook in a background thread.
  • Payload: Title, target companies, geographies, functions, description.

Code: _notify_n8n_project_created

End‑to‑End Flow: Part 2

2) n8n Orchestrates Expert Search

n8n calls back /api/v1/n8n/search-experts.

  • Actor ID: SEARCH_ACTOR
  • Structured payload: currentJobTitles, currentCompanies, locations
  • Processes in small batches to maintain high query relevance.

Execution Logic

  2. App resolves leads_expert_ids

User fills fields; App notifies n8n.

3. Lead Outreach

4. Feedback

Dashboards update based on expert responses.

Ops: Config & Troubleshooting

Configuration

  • N8N_WEBHOOK_URL
  • N8N_SERVICE_TOKEN
  • APIFY_API_TOKEN
  • BREVO_API_KEY

Common Issues

401 Error: Check N8N_SERVICE_TOKEN.

No Results: Check Apify Actor limits.
