System Overview

  • Trigger: New Project creation fires an n8n Webhook.
  • Orchestration: n8n manages data flow between App and Apify.
  • Outreach: Automated invitations to lead experts via Brevo/Resend.

Troubleshooting

  • 401 Unauthorized: Verify N8N_SERVICE_TOKEN match.
  • Mock Results: Check Apify limits; system defaults to mock for demo.
  • Email Issues: Check EMAIL_PROVIDER in flask.env.
